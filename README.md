# MilAssoc Pro

Application de gestion pour association militaire (SPA - Single Page Application).

## Fonctionnalités
- Gestion des membres (CRUD)
- Gestion des événements et calendrier
- Suivi financier (Recettes/Dépenses)
- Gestion documentaire
- Messagerie interne
- Génération de cartes de membre imprimables
- Thèmes personnalisables
