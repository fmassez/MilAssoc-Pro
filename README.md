# MilAssoc Pro - Gestion d'Association Militaire

Application web de gestion pour associations militaires avec interface moderne et responsive.

## ✨ Fonctionnalités

- 📊 Tableau de bord complet
- 👥 Gestion des membres (CRUD)
- 📅 Gestion des événements
- 💰 Suivi financier
- 📄 Gestion documentaire
- 💬 Messagerie interne
- 🏢 Gestion des sections/unités
- ⚙️ Administration configurable
- 🎨 Thèmes personnalisables

## 🚀 Installation

### Option 1 : GitHub Pages (Recommandé)

1. Créez un compte GitHub
2. Créez un nouveau repository
3. Uploadez le fichier `index.html`
4. Activez GitHub Pages dans les paramètres
5. Accédez à `https://votre-nom.github.io/nom-repo`

### Option 2 : Hébergement local

1. Installez un serveur web (Apache, Nginx, ou XAMPP)
2. Placez le fichier `index.html` dans le dossier www/htdocs
3. Accédez à `http://localhost`

### Option 3 : Netlify/Vercel

1. Glissez-déposez le fichier sur [Netlify Drop](https://app.netlify.com/drop)
2. Ou connectez votre repository GitHub

## 💾 Sauvegarde des données

Les données sont stockées dans le **localStorage** du navigateur. Pour exporter :
1. Allez dans Paramètres
2. Cliquez sur "Exporter JSON"
3. Sauvegardez le fichier

Pour importer : utilisez la fonction d'import dans les paramètres.
