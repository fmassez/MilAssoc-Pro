# MilAssoc Pro — Gestion d'Association Militaire

Application web complète de gestion pour associations militaires (type UNC).

## ✨ Fonctionnalités

- 📊 **Tableau de bord** avec statistiques en temps réel
- 👥 **Gestion des membres** (CRUD complet, import/export CSV, feuille d'émargement)
- 📅 **Gestion des événements** avec calendrier
- 💰 **Gestion financière** avec état des comptes imprimable
- 📄 **Gestion documentaire** avec pièces jointes, modification et envoi par email
- 💬 **Messagerie interne**
- 🏢 **Gestion des sections/unités**
- 🪪 **Création de cartes de membre** (format A5/A6, conforme UNC)
- ⚙️ **Administration complète** (grades, sections, types, gestionnaires avec signatures)
- 🎨 **Thèmes personnalisables** (5 sombres + 3 clairs + couleurs manuelles)
- 🖼️ **Logo et favicon** personnalisables

## 🚀 Installation

### GitHub Pages (Recommandé)
1. Créez un repository sur GitHub
2. Uploadez `index.html`
3. Settings > Pages > Source: main
4. Accès : `https://votre-user.github.io/nom-repo`

### Netlify (Plus simple)
1. Allez sur https://app.netlify.com/drop
2. Glissez-déposez le fichier
3. Votre URL est générée automatiquement

### Local
```bash
python -m http.server 8000
# Puis http://localhost:8000
